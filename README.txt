CS184 AS4
by Anthony Tran and Caydie Tran

This is an OpenGL program that runs on OS X platform. It is an IK solver with 4 joints. The user can interact with the program via clicking on the screen. 

The source code is in example_04/example_04/example_04.cpp and other files in that folder. To compile, direct to the outermost example_04 folder and simply run make, followed by ./as4. OR simply input ‘sh as4.sh’ in the same folder as above. 

Anthony Tran will submit the README file, and Caydie Tran will submit the actual code and the README file.
